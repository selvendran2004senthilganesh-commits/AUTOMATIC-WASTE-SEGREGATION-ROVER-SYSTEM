# AUTOMATIC-WASTE-SEGREGATION-ROVER-SYSTEM
Automatic Waste Segregation Rover System is a smart robot that detects and separates waste into wet and dry categories using sensors and a microcontroller. It helps in efficient waste management and reduces manual effort.


## Description
This project is an IoT-based system that separates waste into wet, dry, and metal automatically.

## Features
- Automatic waste detection
- Mobile app monitoring
- Alerts system

## Technologies Used
- Arduino UNO
- ESP32
- Sensors
- Android App
